# Список рядом

«Список рядом» — локальное Android-приложение для ведения постоянного личного списка покупок с собственной базой товаров, магазинами, отделами и локальным хранением данных. Проект развивается локально на ноутбуке, основная IDE — Android Studio, а основная среда агентской разработки — Claude Code.

В репозитории есть специальная проектная база знаний для ИИ-агентов: [`_agent_knowledge`](./_agent_knowledge). Она нужна, чтобы любой агент быстро понимал концепцию проекта, текущий этап, обязательные правила, границы первой версии, уже принятые решения и порядок актуализации контекста после каждой правки.

Эта knowledge-база намеренно сделана tool-neutral: ей должны одинаково пользоваться Claude Code, Codex, Cursor и другие ИИ-агенты, работающие с проектом.

Основные смысловые разделы `_agent_knowledge`:
- обзор проекта и текущее состояние;
- пользовательские правила, правила агента и модель итераций;
- каркас приложения, функциональные требования и UI/UX-ограничения;
- журнал проектных решений, текущий фокус и проектные запреты;
- маршрутизатор знаний, протокол актуализации и changelog knowledge-базы;
- контекст первоисточников и границы достоверности.

Точка входа для агента: [`AGENTS.md`](./AGENTS.md) → [`_agent_knowledge/11_KNOWLEDGE_ROUTER.md`](./_agent_knowledge/11_KNOWLEDGE_ROUTER.md) → [`_agent_knowledge/03_AGENT_RULES.md`](./_agent_knowledge/03_AGENT_RULES.md) → профильные файлы по задаче → [`_agent_knowledge/12_UPDATE_PROTOCOL.md`](./_agent_knowledge/12_UPDATE_PROTOCOL.md) перед завершением итерации.
